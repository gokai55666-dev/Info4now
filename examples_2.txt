🛠️ Core API Parameters: Your Primary Control Tools

For precise, consistent results, moving beyond the basic app interface to use Vidu's API parameters is the most effective strategy. Here are the key parameters to control:

Parameter Value Type Description / Effect
vidu_type string Defines the generation mode: reference2video (reference to video), img2video (image to video), extend (video extension).
duration int Controls video length. For Q2 models, the default is 5 seconds, ranging from 1-10 seconds. For video extension, you can add 1-7 seconds.
seed int Crucial for reproducibility. Setting a fixed integer ensures you get the same result every time, which is essential for iteration and building a consistent visual library.
movement_amplitude string Dictates motion intensity. Values: auto (default), small, medium, large.
first_frame_url / last_frame_url string Required for start-end2video mode. The resolution ratio between the two frames must be between 0.8 and 1.25 for a smooth transition.
moderation boolean This flag defaults to false in some advanced nodes (like ComfyUI). Setting it to true may enable stricter content filtering.
